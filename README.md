## Project Overview

This repository contains a collection of laboratory projects, organized by lab. Each lab folder includes:
RISC-V assembly programs (.s files`) related to the lab exercises.
A Makefile for each assembly code.
Additional files for the visualizer and analyzing the pipeline behavior.
Helper scripts for compilation and simulation using gem5, along with logs and output files for test verification.
